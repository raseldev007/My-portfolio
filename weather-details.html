BD Weather App

Weather Intelligence Platform

Project Overview

BD Weather App is the ultimate weather companion for users in Bangladesh. Designed with a focus on locality, it delivers hyper-local forecasts, real-time alerts, and detailed atmospheric data. Whether you are a traveler, farmer, or daily commuter, this app ensures you are always prepared for the elements.

Key Features

  • Real-time temperature and humidity tracking

  • 7-Day Forecast with rain probability

  • Location-based automated detection

  • Severe weather alerts for storms and cyclones

Technology Stack

Flutter Dart OpenWeather API Google Maps API
