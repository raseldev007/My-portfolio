NBDA – Blood Donation App

NBDA is a blood donation management application designed to connect blood donors with patients and hospitals efficiently. The app helps save lives by enabling fast donor discovery and emergency coordination.

Key Features

  • Real-time donor search
  • Blood group based filtering
  • Location-based donor filtering
  • Emergency blood request system
  • Direct contact with donors

Technology Used

Flutter, Firebase Authentication, Cloud Firestore, Firebase Storage

Amar Bari – Rent Manager

Amar Bari is a Bangladesh-focused rent and tenant management mobile application designed to digitize house rent collection, utility billing, and tenant record management.

Key Features

  • Owner & Tenant role-based system
  • Digital rent collection (bKash/Nagad)
  • Automated monthly rent reminders
  • Tenant document storage (NID/Photo)
  • Utility bill tracking & PDF receipts

Technology Used

Flutter, Firebase Auth, Cloud Firestore, Cloud Functions
