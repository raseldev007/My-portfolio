Amar Bari

Rent & Tenant Management System

Project Overview

Amar Bari is a comprehensive solution for landlords and tenants in Bangladesh. It digitizes the entire rental lifecycle, from tenant onboarding to monthly rent collection. With features like automated reminders and digital receipt generation, it brings transparency and ease to property management.

Key Features

  • Role-based Access (Owner/Tenant)

  • Digital Rent Collection & Receipts

  • Tenant Document Cloud Storage

  • Utility Bill Management

Advanced Management Tools

  • One-Tap Flat Status (Occupied/Vacant)

  • Automated Invoice Generation

  • In-App Support Ticket System

  • Detailed Payment History Logs

Technology Stack

Flutter Firebase Auth Firestore Cloud Functions
